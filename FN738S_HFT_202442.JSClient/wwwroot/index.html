Harry Potter Menu

Harry Potter Full Stack App

This is an individual semester assignment that contains advanced c# knowledge. With this project I learned about the following topics:

  • Entity Framework Core
  • Code-first approach
  • SQL Server database connection
  • Navigation Properties
  • CRUD/NON CRUD Methods
  • Asynchronous programming
  • NUnit
  • HTML/CSS/JS
  • WPF

It's a Harry Potter themed project, you can add Teachers, Students, and Subjects to different Houses and call NON CRUD methods to get back some data. Nunit Moq tests included too.
